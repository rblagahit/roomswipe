# RoomSwipe

**Find Your Perfect Roommate**

A Tinder-style swipe interface for finding compatible roommates. Swipe right if you vibe, left if you don't!

## Features

- 🎨 Beautiful, modern UI with smooth animations
- 📱 Mobile-responsive design
- 💚 Like, Nope, Super Like, and Undo actions
- 🎯 Interactive swipe cards with drag-and-drop
- ✨ Enhanced button animations and visual feedback

## Demo

Open `index.html` in your browser to try it out!

## Tech Stack

- Pure HTML, CSS, and JavaScript
- Google Fonts (Poppins)
- No dependencies or frameworks required

## Deployment

This site can be deployed on GitHub Pages:

1. Go to your repository settings
2. Navigate to **Pages**
3. Select **main** branch as source
4. Your site will be live at: `https://rblagahit.github.io/roomswipe/`
